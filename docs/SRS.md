# Student Management System

## Purpose

The purpose of this application is to allow an administrator to manage student records efficiently.

---

## Functional Requirements

The system shall allow users to:

- Add a student
- Update a student's information
- Delete a student
- Search for a student
- Display all students

---

## Non-Functional Requirements

The system should:

- Be easy to use
- Be reliable
- Provide fast responses
- Have a simple console-based interface

---

## Users

Primary User:

- School Administrator
